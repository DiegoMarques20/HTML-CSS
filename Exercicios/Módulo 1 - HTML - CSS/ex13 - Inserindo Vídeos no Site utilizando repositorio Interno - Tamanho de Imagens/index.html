Inserindo vídeos hospedados internamente

Este vídeo está hospedado no meu própio servidor.

Para incluirmos um vídeo do própio servidor devemos usar a Tag Video, porém dentro dela devemos abrir como Video controls Width='500' poster='imagem/capa.png'

  • Video Controls: Permite aparecer o controle de play e pausa no vídeo
  • Width: Permite você aumentar ou diminuir o tamanho do vídeo no site
  • Poster: Permite você colocar uma capa inicial no seu vídeo

Além dessas informações, assim como imagens e audios, os Vídeos podem ter incopatibilidade de acordo com o navegador que o usuário utilizada, com isso devemos subir vídeos em 3 formatos que são os mais utilizados e obedecendo a ordem MP4,WEBM e OGV. Lembrando que para mencionar os formatos no Type o OGV é informado como OGG.

Utilizamos o souce src para colocar as informações dos vídeos em formatos diferentes e se o navegador não ler um ele tenta a leitura do próximo e assim sucessivamente
