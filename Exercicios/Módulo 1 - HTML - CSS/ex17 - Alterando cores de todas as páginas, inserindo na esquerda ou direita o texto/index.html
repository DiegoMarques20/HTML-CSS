Estilos Externos (Padrão mais utilizado)

Para utilizarmos o estilo externo, uma modificação feita altera todas as páginas de um site, para isso devemos utilizar a TAG link rel="stylesheet" href="style.css"> e inseri esse código em cada página que você deseja ter essas informações padronizadas Em href devemos passar o nome no novo link criado, onde vai conter apenas as informações das cores, letras ou fontes e precisamos informar no cabeçalho desse novo link a seguinte informação (@charset "UTF-8";)

Outro ponto importante é que podemos alterar a centralização das palavras utilizando a TAG (style="text-align:right;") Right é Direita e Left Esquerda

Utilizando a configuração para inserir fonte da letra

Na parte de cima, abaixo de title, inserimos uma Tag Style e dentro dela inserimos o item que queremos modificar a abrimos um colchete {}. Exemplo (style) body {font-family: Arial, Helvetica, sans-serif;} (/style)

Utilizando a configuração para inserir tamanho da letra

Na parte de cima, abaixo de title, inserimos uma Tag Style e dentro dela inserimos o item que queremos modificar a abrimos um colchete {}. Exemplo (style) body {font-size: 15px;} (/style)
