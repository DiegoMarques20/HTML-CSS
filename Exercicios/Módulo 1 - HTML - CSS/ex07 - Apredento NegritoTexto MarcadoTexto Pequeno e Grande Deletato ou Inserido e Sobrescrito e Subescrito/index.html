Principais formatações

Negrito / Destaque

Nessa frase temos um termo em Negrito Para isso utilizamos o <b> e fechamos com </b> (Forma não semântica)

Já nesta frase temos também um termo em destaque, porém utilizando a forma semântica do STRONG

Agora vamos ter um termo em italico usando a tag <i> não semântica

E agora utilizando italico com semântica da tag EM

Texto Marcado

Podemos realizar uma marcação similar ao marca texto utilizando a tag MARK

Para mudarmos a cor em uma marcação por exemplo, pois o site pode não combinar com o amarelo, devemos atuar em CSS utilizando a função STYLE, como neste exemplo

Para não precisarmos sempre alterar cores de vários itens, podemos fazer um seletor da tag STYLE em cima do cabeçalho acima de head

Texto Pequeno e Texto Grande

Para aumentar uma letras, embora essa tag vai ser descontinuada, pois podemos usar o CSS para formatação, ainda podemos usar o BIG

Já para diminuirmos a letra, podemos usar a tag SMALL, como nesse exemplo: o cruzeiro é um time pequeno

Texto Deletado e Inserido

Podemos marcar um texto que já foi excluido, utilizando a tag DEL

Para um texto que foi inserido e para indicar que foi adicionado posterioremente utilizamos a tag INS

Referente ao exemplo acima, podemos utilizar a tag U, porém ela é uma forma não semântica

Texto Sobrescrito e Subescrito

Podemos inserir coisas do tipo X20+3 para sobrescrito, usando a tag SUP

E para subescrito itens do tipo H20, utilizamos a tag SUB
