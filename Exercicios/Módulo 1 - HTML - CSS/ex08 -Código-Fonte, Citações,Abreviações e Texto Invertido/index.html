Exemplo da utilização de um código Python representado abaixo, usamos a tag Code para modificar a estrutura da letra e a tag Pre que faz a indentação para visualmente os códigos aparecem formatados um abaixo do outro

        
    num= int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é IMPAR')
    print('Fim do programa')

Para deixaramos o código para o lado ou para o outro, devemos selecionar todo o código e aperta Shift + Tab ou apenas TAB.
