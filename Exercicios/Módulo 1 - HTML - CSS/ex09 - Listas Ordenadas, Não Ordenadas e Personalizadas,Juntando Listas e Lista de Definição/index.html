Personalização de listas

Podemos personalizar as listas, conforme no exemplo abaixo utilizando dentro da tag OL type = A (A lista é enumerada iniciando do a,b,c) type = I (Lista enumera em algarismos romanos) type=1 start="4" (A enumeração inicia contando com o número 4
