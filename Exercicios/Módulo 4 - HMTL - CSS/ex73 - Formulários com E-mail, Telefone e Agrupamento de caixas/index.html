Formulário

  • Para solicitarmos um e-mail, usamos o input email. Inserimos o autocomplete="email" para que a caixa de preenchimento não deixe o usuário preencher um e-mail sem o @.

  • Para informações de telefone, usamos o input tel, e para configurarmos que o usuário digite apenas números e em quantidade específica, utilizamos pattern="^\(\d{2}\)\d{4,5}-\d{4}$". Depois pesquisar mais sobre RegEx em HTML.

  • Para agruparmos o formulário, usamos a tag fieldset, com isso os dados são agrupados dentro de um formato quadrado. Para darmos nome para esse agrupamento, usamos a tag legend.
