Utilizando Get e Post

  • Quando criamos um formulário de maneira simples, os dados digitados pelo usuário aparcem na url da próxima página.

  • Quando esses dados utilizamos pelo usuário não forem dados sensíveis, utilizamos o metódo Get, dentro da tag Form inserimos method="get"

  • Quando tivermos que utilizar dados sensíveis do usuário, dados com mais de 3000 bytes (3000 letras), envio de arquivos ai neste caso utilizamos o Post , method="post", dentro da tag form.
