Formlários

  • Para iniciarmos a criação de um formulário, utilizamos a tag form.

  • Para criarmos aquelas caixas de preenchimento, inserimos o input, neste caso vamos fazer inserir um texto, então usamos input type="text" e informamos o nome ou id, conforme nesse exemplo: input type="text" name="nome" id="nome"

  • Para inserimos o botão de enviar dentro de um formulário , usamos o o input do tipo submit e dentro de value, passamos o nome da que desejamos que apareça dentro do botão. input type="submit" value="Enviar">
