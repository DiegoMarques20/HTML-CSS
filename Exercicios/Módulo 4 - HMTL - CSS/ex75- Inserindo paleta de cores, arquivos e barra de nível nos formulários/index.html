Exemplo de Formulário

  • Para abrirmos um seletor de cores, utilizamos o input color.Se quizermos passar uma cor em específico utilizamos o value="" e inserimos o código da cor.

  • Para criarmos uma barra de nível, como no exemplo abaixo, utilizamos o input range, para personalizarmos podemos inserir a numeração máxima e mínima, utilizando max e min.

  • Para envio de arquivos, utilizamos o input file, sempre que utilizamos esse input, o method precisa ser post.
