Criando caixas de texto e de senha

  • Para criamos uma caixa de senha, onde os números digitados pelo usuário não aparecem, utilizamos o input type="password", e identificamos o nome para vincular ao texto senha, por exemplo.

  • Para criarmos um botão de limpar os dados escritos pelo usuário, utilizamos o input type="reset" value="Limpar"

  • Quando criamos um formulário, precisamos que o usuário digite algo para ele enviar, evitando que ele envie informações em branco, para esses casos, inserimos dentro da tag input a palavra required, com isso caso o usuário deixe algum campo em branco, ele vai aparecer uma mensagem informando para digitar as informações antes do envio.

  • Podemos especificar também a quantidade mínima e máxima de caracteres que o usuário precisa digitar, neste caso usamos dentro da tag input minlength="5" e maxlength="15", neste exemplo o usuário precisa de no mínimo 5 caracteres digitados e no máximo para ser validado.

  • Para organizarmos o tamanho da caixa de digitação, usamos a tag size dentro do input, neste exemplo estamos utilizando size="10".

  • Para deixarmos um texto discreto dento da caixa de digitação, usamos a tag placeholder, neste exemplo utilizamos placeholder="nome de usuário"

  • Podemos usar o autocomplete para deixar salvo no navegador os dados digitados pelo usuário, neste exemplo dentro do form autocomplete="on" e dentro do input inserimos autocomplete="username", Username significa usuário, temos outras possibilidade como autocomplete="current-password", salvaria o senha do usuário.
