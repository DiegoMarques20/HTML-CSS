Mobile First

  • Mobile first significa mobile primeiro, é uma técnica que criamos a responsividade mobile primeiro e sites leves para depois ajustar os demais itens, algumas vantagens de utilizar essa técnica são:

  • Site de buscas reconhecem melhor
  • Site mais acessível, pois não fica muito pesado e usúarios com velocidade baixa de internet conseguem acessar normalmente.
  • Boa experiência do usúario.

  • No próximo exemplo, vamos configurar as midias queries separadas pela configuração geral e pelas específicas, criando dois links de CSS. A primeira tudo que for geral e a segunda vamos inserir todas as configurações para dispositivos diferentes.
