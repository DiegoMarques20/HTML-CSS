Configurando botões do input

  • Para configurarmos um botão que está em um input, realizamos desta forma form > input[type=submit]. Informos dentro de onde está inserido o input, nesse exemplo dentro de um form, e entre colchetes passamos o nome do tipo, neste caso o tipo foi submit

  • Para usarmos o hover, a situação é a mesma, segue exemplo: form > input[type=submit]:hover.
