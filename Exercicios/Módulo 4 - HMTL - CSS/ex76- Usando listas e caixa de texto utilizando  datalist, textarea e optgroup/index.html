Formulários

  • Para selecionarmos uma lista em forma de estados por exemplo, utilizamos o select, criamos um label e passamos o mesmo name ou id, e dentro do select criamos categorias utilizando o option, que são as opções selecionadas pelo usuário. Além disso podemos criar sub grupos, como exemplo de seletories de estados, os subgrupo de região sudeste, nordeste, sul e etc. Para isso envelopamos o option utilizando o optgroup e dentro do optgroup informamos a label com o nome desejado para destacar o subgrupo, como no exemplo abaixo.

  • Para deixarmos uma msg dentro do seletor, como escolha seu estado ou algo similar, usamos o option value=""> --Escolha Seu Estado-- option.

  • Para Criarmos uma lista similar, porém onde o usuário pode digitar para encontrar sua opção, utilizamos o input text e label normalmente e criamos uma tag datalist e informamos um nome de id e dentro dela inserimos option que são as opções disponiveis, com isso ativamos dentro da tag input text list e passamos o nome da lista criada.

  • Para criamos uma caixa de texto, utilizamos a tag textarea e dentro dele passamos o nome da label como de costume e a quantidade de números da largura e altura da caixa, como padrão ele vem com cols=30 e rows=10. Para esse metódo devemos utilizar method="post".
