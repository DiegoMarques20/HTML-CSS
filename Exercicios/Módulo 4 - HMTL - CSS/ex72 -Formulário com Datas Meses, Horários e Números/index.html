Teste de Formulário

  • Dentro de um formulário, se desejarmos solicitar apenas números, usamos o input number, pois ele permite que o usuário digite apenas números. Para padronizarmos a quantidade mínima e máxima de números digitados, repassamos dentro do input min="0" max="10"

  • Se quizermos números padronizados, como notas que podem ter valores 5,5, 9,6 inserimos um step="0.1"

  • Para datas, podemos utilizar o input month ou input date

  • Para hora, utilizamos o input time.
