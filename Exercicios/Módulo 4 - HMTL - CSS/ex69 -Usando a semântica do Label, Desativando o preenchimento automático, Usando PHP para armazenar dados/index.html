Melhorando os Formulários

  • Normalmente o preenchimento de formulário é auto completado, com nomes que já utilizamos outras vezes no navegador, para desativarmos essa opção inserimos dentro da tag form o autocomplete="off"

  • Para enviarmos as informações prrenchidas para armazenamendo, precisamos utilizar uma linguagem de programação e usamos o action dentro da tag form, como neste exemplo que estamos simulando o uso da linguagem PHP form action="cadastro.php".

  • Utilizando o Label: Devemos sempre utilizar o label, pois ele faz ligação entre o componente nome e a caixa onde o usuário vai digitar, pois para os mecanismos de busca é extremamente importante essa ligação entre o texto e a caixa de digitação, outro ponto é que para usuários que utilizam o celular e precisam selecionar com o dedo um campo para digitar, o label melhora o alçance do click.

  • Para fazer essa ligação informada acima, utilizamos a tag Label no nome da caixa utilizada, nesse nosso exemplo usariamos em nome e sobrenome e depois de aplicar a tag, passariamos dentro do for o nome de id (JavaScrip) ou name (PHP), para cada liguagem de programação um ou outro é mais importante, conforme parênteses.
