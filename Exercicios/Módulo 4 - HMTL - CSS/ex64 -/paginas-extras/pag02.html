Testando uso de Iframe

  • Alguns dispositivos como de celulares, não são compatíveis com o iframe, neste caso add a tag A e informamos o link. pois em caso de imcompatibilidade o link do site do iframe é informado.

  • O tamanho padrão de um iframe é de 300x150, mas caso precisamos alterar inserimos os novos valores utilizando width e height dentro da tag iframe.

  • O frameborder dentro da tag iframe, já vem padronizado com 0, porém podemos incluir quaqluer valor acima de 0 que ele criar uma borda em volta do site inserido no iframe.
