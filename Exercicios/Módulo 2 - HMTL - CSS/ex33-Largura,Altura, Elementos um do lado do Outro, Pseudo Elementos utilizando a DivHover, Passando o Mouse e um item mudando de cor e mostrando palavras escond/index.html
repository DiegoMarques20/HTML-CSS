Pseudo Classes

Neste exemplo vamos criar algumas Divs e dentro dela vamos fazer algumas ações, como se o usuário passar o mouse alguma ação vai acontecer. para representar as Pseudo classes, utilizamos os dois pontos : , por exemplo a pseudo class da div:houver é uma configuração de estado, então se usamos div:hover {background-color: yellow;} quando o usuário passar o mouse em alguma div o mesmo vai ficar amarelo.

Altura, largura, Elementros um do lado do outro e Borda

Para alterarmos a altura utilizamos a tag height: 200px; e para largura utilizamos width: 200;.E para deixarmos elementos um do lado do outro é display: inline-block; .Para inserirmos borda utilizamos border: 1px solid black;
