  • Identificamos pseudo class, pois essas precisam de obter : como exemplo a:hover e pseudo elementos possuem :: como a::after

  • Para retirarmos o sublinhado dos link, dentro de style usamos a tag text-decoration: none;

  • Para alterarmos a cor de um link que o usúario já acessou, igual acontece que utilizamos buscas no google, usamos a pseudo class a:visited e inserimos a cor desejada, conforme nesse exemplo: a:visited{ color: blueviolet;}

  • Para o usuário passar o mouse em cima de uma informação e ficar sublinhado, utilizamos a pseudo class a:hover da seguinte maneira: a:hover{ text-decoration: underline; }

  • Para indicarmos um link a frente de uma palavra/link usamos o pseudo elemento ::after, como neste exemplo: a::after{ content: ' [Link]';

  • Para inserirmos símbolos antes ou depois, buscamos no google símbolos HTML 5 cópiamos o código do símbolo ou a propria imagem e colamos conforme utilizamos os pseudos elementos abaixo:

    • Antes do link: a::before{ content: '⇒ ' ;

    Depois do link: a::after{ content: ' ⇐ ';
