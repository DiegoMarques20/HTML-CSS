Seletores personalizados

Para personalizarmos um h1 ou p ou h2, normalmente quando modificávamos uma fonte ou cor todos se modificavam, agora para alterarmos a configuração de um específico temos que nomer com id Exemplo dentro do <> colocamos o nome que quisermos id="prinpical" e quando formos alterar inserirmos uma # e exatamente o nome da id e dentro do parênteses {} inserimos as configurações personalizadas.

Pontos Importantes

Lembrando que o nome precisa estar exatamente igual, pois se tiver algo de diferente o código não funciona. Em HTML id é igual a # em CSS. Em HTML class é igual a .
