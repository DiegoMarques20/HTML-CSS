Utilizando Class

Para usar ID só podemos usar um ID com o mesmo nome, e se quizermos fazer alterações pontuais para um mesmo grupo usamos a tag class e demos um nome para ela dentro do <> que desejamos.Exemplo: class = "primeiradivisao" e na hora de informamos isso dentro no CSS ficaria .primeiradivisao. Com isso toda class que informamos 1divisao ela vai estar com configuração padronizada, como no exemplo neste texto.
