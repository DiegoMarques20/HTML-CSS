Site com Desig interessante

  • Para criarmos um link apenas para CSS, abaixo de Title, inserimos o código link rel=stylesheet href=style.css e abrimos as configurações do CSS em outro index.

  • Aberto o outro index do CSS, inseremos a confiduração @charset"UTF-8'; para o site poder conter caracteres especiais

  • Para fonte do nosso texto, vamos utilizar o font-family: Arial, Helvetica, sans-serif

  • E para alinharmos o texto, vamos utilizar para os paragráfos e Li o text-align: justify.

Edição de Cores e Configurações

Dentro do link das configurações do CSS, precisamos realizar algumas configurações que são importantes como no exemplo abaixo.

  • background-color: white; Para definir a cor

  • width: 600px; Para definir o tamanho

  • padding: 10px; Define o espaço entre o conteúdo e as bordas do canto

  • border-radius: 20px; Deixa as bordas do conteúdo arredondadas

  • box-shadow: 5px 5px 15px #9e8173a0; Insere uma sobra no conteúdo as numerações são a escala do sombreamento e a cor é o entorno

  • margin: auto; Centraliza os elementos do conteúdo

  • text-shadow: 1px 1px 0px #9e8173a0; Faz o mesmo sombreado do box, porém esse é específico para textos
