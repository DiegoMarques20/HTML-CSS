Inserindo Listas uma ao lado da Outra e Lista para dentro

  • Para inserimos uma lista ao lado da outra, conforme o exemplo, usamos a tag columns: 2. Onde 2 é o número de vezes que queremos que a lista seja dividida.

  • Para deixarmos a lista para dentro do bloco, usamos a tag list-style-position: inside.

Para usarmos símbolos ou emoji dentro de uma lista, usamos a tag list-style-type: '\1F4CD';. E inserirmos uma contra barra e depois o códgo do símbolo selecionado.
