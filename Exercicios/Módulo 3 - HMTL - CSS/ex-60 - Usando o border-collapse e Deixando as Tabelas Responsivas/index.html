Tabelas Responsivas

  • Para deixarmos a tabela em forma de tabela tradicional add a tag border-collapse: collapse;

  • Para deixarmos a tabela responsiva, selecionamos todo conteúdo dentro de table e envelopamos inserindo uma DIV, inserimos um nome de id da div e utilizamos a tag overflow-x: auto;, neste caso a tabela já está responsiva para caso o usuário abrir em um celular.
