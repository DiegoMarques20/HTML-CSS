Deixar o Cabeçalho de uma tabela Fixo, efeito congelar painéis no Excel

  • Para fixarmos o cabeçalho de uma tabela, como neste exemplo utilizamos as sequência de acordo com hierarquia de tabelas (thead > tr > th) e usamos a tag position: sticky; junto com top: -1px; e depois inserimos a mesma cor do cabeçalho usando um background-color:gray;, conforme usamos neste exemplo.


    thead > tr > th {

    position: sticky;

    top: -1px;

    background-color:gray;
    }
