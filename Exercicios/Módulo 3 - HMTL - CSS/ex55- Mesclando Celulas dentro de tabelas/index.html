Mesclagem de Células

  • Para mesclarmos uma célula, conforme fizemos com B, usamos colspan="2" dentro da tag td que é o dado que queremos mesclar, e o número 2 signifca a quantidade de colunas que vamos mesclar, nesse caso serão 2

  • Para ocuparmos mais de uma linha, conforme fizemos com o D, usamos o rowspan="3" dentro da tag td, 3 significa o número de linhas ocupadas.
