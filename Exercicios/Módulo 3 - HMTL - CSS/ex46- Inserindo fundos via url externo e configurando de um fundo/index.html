Inserindo fundos via url externo e configurando um fundo

  • Podemos inserir um fundo de uma fonte externa, basta passarmos o endereço url e inserir dentro da tag background-image: url('https://gustavoguanabara.github.io/html-css/imagens/mascote.png'); Lembrando que a imagem precisa estar configurada em png,jpeg etc.

  • Podemos configurar o tamanho do fundo usando a tag background-size: 100px 100px; Os numeros em px determinam o tamanho.

  • Podemos configurar o fundo para não repetir a imagem de fundo e ela ficar apenas em um bloco usando background-repeat: no-repeat; Ou podemos inserir a imagem de fundo em apenas no eixo X ou Y, para ficar apenas na lateral ou na horizntal, nestes casos utilizamos o background-repeat: repeat-x; ou background-repeat: repeat-y;
