Efeito Zebrado

  • Para utilizarmos o efeito zebrado, usamos a pseudo class desta forma tbody > tr:nth-child(2n) O número 2 representa a quantidade de linhas que vão ficar marcadas, ou seja pulando de dois em dois, se quiser outro número só add entre os parênteses.

  • Podemos colocar também dentro do 2n no exemplo usado a cima, par ou ímpar, neste caso ele destaca as linhas pares ou ímpares de acordo com o selecionado. tbody > tr:nth-child(odd) ou tbody > tr:nth-child(even)

  • Se usarmos aguma númeração como utilizamos em populacão, dentro do td, colocamos class="num" dentro da tag.

  • Para modificarmos a largura ou altura de uma coluna, dentro do th informamos a quantidade de pixels desta forma style="width:200px" ou style="height: 200px".
