Teste imagem de Fundo Configurando Visualização

  • Pode ocorrer de uma imagem de fundo ficar desconfigurada, para isso podemos usar a alternativa do background-size: cover; ou background-size: 100%; Vai depender do objetivo e teste do site com a imgem.

  • Uma outra boa opção é usarmos o background-size: cover; e background-attachment: fixed; para que o site mude de acordo com o dispositivo que o cliente está acessando.

  • Podemos resumir o uso do background na sequência Color > Image > Position > Repeat > Size > Attachment
