Inserindo Imagens no Fundo

  • Para inserimos imagens no fundo, podemos criar uma DIV e dentro dela fazer bordas específicas e usar a tag background-color: para selecionar uma cor de fundo.

  • A tag background-image: linear-gradient(to bottom, yellow,red); para selecionar cores de forma degradê do fundo

  • Ou podemos usar a tag background-image: url(imagens/pattern003.png); neste caso pegamos uma imagem externa inserimos na pasta e dentro de url colocamos o nome do arquivo desejado, conforme neste fundo de exemplo.
